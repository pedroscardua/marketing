# Dicas de Execucao

1. **Horarios:** Os horarios sugeridos sao baseados em engajamento tipico. Ajuste conforme o comportamento do seu grupo.

2. **Audios:** Grave de forma natural, como se estivesse falando com uma amiga. Nao leia, use o roteiro como guia.

3. **Respostas:** Responda TODAS as mensagens do grupo. Engajamento gera engajamento.

4. **Screenshots:** Borre dados sensiveis. Use dados fictícios que parecam reais.

5. **Numeros:** Os numeros de beta testers (68% reducao no-show, R$12.800 recuperados) devem ser ajustados para refletir resultados reais.

6. **Urgencia real:** A oferta de fundadora DEVE encerrar no dia 30. Nao estenda. Urgencia falsa destroi credibilidade.

7. **Pos-lancamento:** No dia 01/05, transforme o grupo em canal de suporte das fundadoras. Continue postando conteudo de valor.

8. **Flexibilidade:** Se um conteudo gera muita discussao, permita que a conversa flua. Nao interrompa engajamento organico para postar o conteudo do dia.
