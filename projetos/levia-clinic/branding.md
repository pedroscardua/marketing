## Paleta de Cores

| Nome | Hex | Uso |
|------|-----|-----|
| **Sage Green Profundo** | `#2D4A3E` | Background principal, elementos de marca |
| **Sage Green Escuro** | `#243D33` | Variação de fundo para CTAs e barras |
| **Sage Green Sutil** | `#3A5C4E` | Watermarks, overlays, detalhes secundários |
| **Ivory** | `#F5F0E8` | Tipografia principal, painéis, brand bar |
| **Rose Gold** | `#C9A882` | Acentos, bullets, bordas de círculos, CTAs, regras decorativas |
| **Ivory Translúcido** | `#F5F0E8` a 92–97% | Containers de preço, painéis frosted |

### Regras de uso de cor
- Sage green é sempre o fundo dominante — nunca fundo branco ou azul
- Rose gold é acento, não cor dominante — usado com parcimônia
- Nunca usar neon, azul vibrante, roxo ou cores não listadas
- Glow/luz quente ivory no canto superior direito: máximo 8% opacidade

## Tipografia

### Hierarquia

| Nível | Estilo | Uso |
|-------|--------|-----|
| **Display / Hero** | Ultra-bold condensed sans-serif, tracking -10 a -20 | "COMBO", "BOTOX", títulos principais |
| **Script / Acento** | Cursiva fluida / script elegante | "cuidado exclusivo", subtítulos poéticos |
| **Corpo / Serviço** | Small caps serif ou bold condensed, tracking +50 a +200 | Nomes de procedimentos, labels |
| **Detalhe / Apoio** | Thin italic serif | Subtítulos secundários, "3 regiões", "condição especial" |
| **Marca** | Small caps, tracking +400 a +500 | "LEVIÁ CLINIC" em todas as peças |
| **Caption** | Thin italic, tamanho pequeno | "Dras. Scardua & Fanelli", localização |

### Cores de tipografia
- Títulos hero: **ivory** `#F5F0E8`
- Script/acento: **rose gold** `#C9A882`
- Texto dentro de painéis ivory: **sage green** `#2D4A3E`
- Marca e captions no brand bar: **sage green** `#2D4A3E`

## Elementos Visuais Recorrentes

### Fotos circulares
- Bordas: rose gold `#C9A882`, 3px stroke
- Outer glow: ivory sutil, blur 12px, opacidade 20%
- Conteúdo típico: close-up de lábios / rosto de mulher 35-45 anos
- Estilo fotográfico: editorial beauty, luz studio quente, pele natural

### Regras decorativas (rules)
- Horizontais: rose gold, 1px, usadas como separadores de seção
- Flanqueando "LEVIÁ CLINIC" no brand bar: rose gold, 30-40px cada lado
- Tracejadas: ivory, para separar serviços dentro de listas

### Watermark de fundo
- Forma: folha abstrata, gota ou silhueta floral
- Cor: sage green mais claro `#3A5C4E`
- Opacidade: 8–12%
- Posição: centralizada atrás do elemento hero

### Bullets de lista
- Círculo preenchido rose gold, 8px diâmetro
- Nunca usar traço, asterisco ou emoji como bullet

### Anotações tipo diagrama médico
- Linha: ivory, 0.8px, formato L (horizontal + vertical)
- Dot de contato: círculo ivory preenchido, 4px
- Texto: uppercase, thin sans-serif, 12-13pt, tracking 150

## Containers e Botões

### Container de preço
```
Rounded rectangle
Background: ivory #F5F0E8 (97% opacidade)
Border: sage green #2D4A3E, 1.5px
Border-radius: 16px
Padding: 28-32px vertical · 36px horizontal
```

### Botão CTA (WhatsApp / Agendar)
```
Fill: rose gold #C9A882
Border-radius: 12px
Height: 64-72px
Ícone WhatsApp ivory à esquerda
Texto: ivory bold sans-serif 22-24pt, centralizado
```

### Brand Bar (rodapé obrigatório)
```
Background: ivory #F5F0E8
Border top: sage green 1px
Altura: 50-80px (feed) · 80px (stories)
Esquerda: pin icon + localização (thin sage green)
Centro: "LEVIÁ CLINIC" small caps + regras rose gold laterais
Direita: "Dras. Scardua & Fanelli" thin italic sage green
```

## O Que Nunca Fazer

| Proibido | Alternativa correta |
|---|---|
| Fundo azul, branco ou bege genérico | Sage green `#2D4A3E` |
| Tipografia bold genérica amarela/vermelha | Ivory + rose gold na hierarquia correta |
| Badge "Menor Preço do Brasil" ou similares | Destacar parcelamento com elegância |
| Emoji como bullet point | Círculo rose gold preenchido |
| Sombras pesadas ou neon glows | Glow ivory sutil máximo 20% opacidade |
| Muitos elementos competindo | Hierarquia clara — um elemento hero por zona |
| Tom corporativo ou clínico frio | Elegante mas próximo — premium com calor |
| Logotipo de terceiros em destaque | Só "LEVIÁ CLINIC" + "Dras. Scardua & Fanelli" |

## Checklist antes de gerar qualquer prompt

- [ ] Fundo sage green `#2D4A3E` definido

- [ ] Rose gold usado só como acento
- [ ] Brand bar ivory no rodapé com as 3 seções
- [ ] Fotos circulares com borda rose gold (se aplicável)
- [ ] Container de preço ivory com borda sage green (se aplicável)
- [ ] Botão CTA rose gold com ícone WhatsApp (se aplicável)
- [ ] Nenhuma cor fora da paleta oficial
- [ ] Texto dentro das zonas seguras do formato
